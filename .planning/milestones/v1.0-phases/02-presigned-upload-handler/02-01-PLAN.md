5. **Function: `mapZodError(err: ZodError): ValidationError`** (per D-12):
Extract first issue. Format: `"${issue.message} at '${path}'"` where path is `issue.path.join('.')` or `'(root)'` if empty. Attach raw ZodError as `cause`. Do NOT prepend "Invalid input:" â€” Zod v4 `issue.message` already includes it.

6. **Function: `validateUploadOptions(raw: unknown): UploadOptions`**:
Call `UploadOptionsSchema.parse(raw)`, catch `ZodError`, rethrow via `mapZodError`.

7. **Function: `base64ToUint8Array(b64: string): Uint8Array`**:
Use `globalThis.atob(b64)` to decode, then create `Uint8Array` from char codes. No `Buffer` import.

8. **Function: `materializeStream(stream: ReadableStream<Uint8Array>): Promise<Uint8Array>`**:
Use `stream.getReader()`, read chunks in a loop, concatenate into single `Uint8Array`. Call `reader.releaseLock()` in `finally`.

9. **Function: `readFilePath(path: string): Promise<Uint8Array>`** (per D-04):
Check `typeof process !== 'undefined' && process.versions?.node !== undefined` for Node, `typeof Deno !== 'undefined'` for Deno, `typeof Bun !== 'undefined'` for Bun. If none, throw `new ValidationError("File path input requires Node.js, Deno, or Bun. Pass Buffer or Uint8Array on edge runtimes.")`. Otherwise, `const { readFile } = await import('fs/promises'); const buffer = await readFile(path); return new Uint8Array(buffer.buffer, buffer.byteOffset, buffer.byteLength);`.

10. **Function: `toUint8Array(input: FileInput): Promise<Uint8Array>`** (per D-01, D-03):
- `input instanceof Uint8Array` -> return input
- `input instanceof ReadableStream` -> return `materializeStream(input)`
- `typeof input === 'string'`:
  - starts with `'data:'` -> strip prefix with `input.replace(/^data:[^,]+,/, '')`, decode via `base64ToUint8Array`
  - matches `/^[A-Za-z0-9+/]+=*$/` AND `input.length > 256` -> raw base64, decode via `base64ToUint8Array`
  - else -> file path, call `readFilePath(input)`

11. **Function: `detectContentType(bytes: Uint8Array): SupportedContentType`** (per D-05):
- `bytes.length < 4` -> throw `new ValidationError('File is too small to detect content type (minimum 4 bytes).')`
- JPEG: `bytes[0] === 0xFF && bytes[1] === 0xD8 && bytes[2] === 0xFF` -> `'image/jpeg'`
- PNG: `bytes[0] === 0x89 && bytes[1] === 0x50 && bytes[2] === 0x4E && bytes[3] === 0x47` -> `'image/png'`
- WebP: `bytes.length >= 12 && bytes[0] === 0x52 && bytes[1] === 0x49 && bytes[2] === 0x46 && bytes[3] === 0x46 && bytes[8] === 0x57 && bytes[9] === 0x45 && bytes[10] === 0x42 && bytes[11] === 0x50` -> `'image/webp'`
- else -> throw `new ValidationError('Unsupported image format. Accepted formats: JPEG, PNG, WebP.')`

Export all public types and functions: `FileInput`, `SupportedContentType`, `UploadOptions`, `PresignResponse`, `toUint8Array`, `detectContentType`, `mapZodError`, `validateUploadOptions`. Keep `base64ToUint8Array`, `materializeStream`, `readFilePath` as non-exported internal helpers.
