**compare() method (FACE-02, D-02 — batch presign with count:2, parallel PUTs):**
4-step flow:
1. `FaceCompareInputSchema.parse(input)` with Zod error mapping
2. `this.uploader.upload([validated.source, validated.target])` — ARRAY of 2 inputs triggers batch presign with `count: 2` and parallel S3 PUTs via FileUploader
